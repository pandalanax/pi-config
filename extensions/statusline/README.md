# Custom statusline

This is a vendored customization of [`@narumitw/pi-statusline` 0.14.1](https://www.npmjs.com/package/@narumitw/pi-statusline).

It renders Codex usage (`codex-usage`) and file-change counts (`filechanges`) as segments in the main powerline row instead of a separate extension-status row. The customized default row omits token, cost, and clock segments to keep those usage segments visible.
